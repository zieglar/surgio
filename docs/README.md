### 马上开始

```bash
# 安装
npm init surgio-store my-rule-store

# 或 使用国内镜像安装
npm init surgio-store my-rule-store --use-cnpm

# 生成一批规则
cd my-rule-store
npx surgio generate
```

:::warning 注意
目前 Surgio 仅支持 Node.js 版本 >= 18.0.0。
:::
